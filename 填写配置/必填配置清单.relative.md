# 必填配置清单

你需要填写两类内容：

```text
1. 模型服务和飞书 webhook
2. 抖音账号列表
```

## 一、模型服务和飞书 webhook

正式配置文件路径：

```text
.env
```

可以从这个模板复制：

```text
填写配置/env.template
```

需要填写：

```text
LLM_API_KEY=你的模型服务 API Key
LLM_MODEL=你的模型名
LLM_BASE_URL=你的模型服务 /v1 地址
FEISHU_WEBHOOK=你的飞书机器人 webhook
AI_NEWS_CONFIG=config.yaml
```

示例，DeepSeek：

```text
LLM_API_KEY=sk-xxx
LLM_MODEL=deepseek-chat
LLM_BASE_URL=https://api.deepseek.com/v1
FEISHU_WEBHOOK=https://open.feishu.cn/open-apis/bot/v2/hook/xxx
AI_NEWS_CONFIG=config.yaml
```

## 二、抖音账号列表

正式配置文件路径：

```text
config/douyin_accounts.local.yaml
```

可以从这个模板参考：

```text
填写配置/douyin_accounts.template.yaml
```

每个账号需要填写：

```text
name：账号名称
profile_url：抖音账号主页链接
enabled：true 表示启用
```

`sec_uid` 可以先空着。

## 三、可选配置

如果要改联网检查间隔，编辑：

```text
config.yaml
```

修改：

```yaml
auto_run:
  check_interval_seconds: 1800
```

如果要改关键词，也在 `config.yaml` 里修改：

```yaml
keywords:
  - AI
  - 人工智能
  - 大模型
```

## 四、填完后的检查命令

```powershell
cd "AI News"
python run.py healthcheck
python run.py dry-run --delivery
python run.py dry-run --collector douyin
```
